# Path Scanner

أداة TypeScript إنتاجية لمسح المجلدات والملفات مع دعم كامل للغة العربية.

## المميزات

- ✅ مسح شامل للمجلدات والملفات
- ✅ دعم كامل للغة العربية في الواجهة
- ✅ خيارات متقدمة للتصفية (ملفات فقط، مجلدات فقط)
- ✅ إحصائيات مفصلة (عدد الملفات، المجلدات، الحجم الكلي)
- ✅ حفظ النتائج في ملف
- ✅ معالجة الأخطاء المتقدمة
- ✅ أداء عالي مع قياس الوقت المستغرق

## التثبيت

```bash
# تثبيت التبعيات
npm install

# أو باستخدام pnpm
pnpm install
```

## الاستخدام

### التطوير (تشغيل مباشر)

```bash
# مسح مجلد معين
npm run dev "H:\the-copy\external"

# مسح الملفات فقط
npm run dev "H:\the-copy\external" --files-only

# مسح المجلدات فقط
npm run dev "H:\the-copy\external" --dirs-only

# تضمين الملفات المخفية
npm run dev "H:\the-copy\external" --include-hidden

# حفظ النتائج في ملف
npm run dev "H:\the-copy\external" --output paths.txt
```

### الإنتاج

```bash
# بناء المشروع
npm run build

# تشغيل النسخة المبنية
npm start "H:\the-copy\external"

# أمثلة متقدمة
npm start "H:\the-copy\external" --files-only
npm start "H:\the-copy\external" --output paths.txt
npm start "H:\the-copy\external" --dirs-only --include-hidden
```

## الخيارات المتاحة

| الخيار | الوصف |
|--------|--------|
| `--files-only` | عرض الملفات فقط |
| `--dirs-only` | عرض المجلدات فقط |
| `--include-hidden` | تضمين الملفات والمجلدات المخفية |
| `--output <ملف>` | حفظ النتائج في ملف |
| `--help`, `-h` | عرض المساعدة |

## أمثلة الاستخدام

### مسح شامل لمجلد

```bash
npm run dev "H:\the-copy\external"
```

### مسح الملفات وحفظها

```bash
npm run dev "H:\the-copy\external" --files-only --output all-files.txt
```

### مسح المجلدات مع الملفات المخفية

```bash
npm run dev "H:\the-copy\external" --dirs-only --include-hidden
```

## الإحصائيات

الأداة تعرض إحصائيات مفصلة تشمل:

- **عدد المجلدات**: إجمالي المجلدات الممسوحة
- **عدد الملفات**: إجمالي الملفات الممسوحة
- **الحجم الكلي**: الحجم الإجمالي للملفات (بالبايت، كيلوبايت، ميجابايت، إلخ)
- **المسارات المعروضة**: عدد المسارات المطابقة للفلتر
- **الوقت المستغرق**: الوقت المستغرق في المسح

## معالجة الأخطاء

الأداة تتعامل مع الأخطاء بشكل متقدم:

- عرض الأخطاء مع المسارات المحددة
- استمرار المسح حتى في حالة وجود أخطاء
- رسائل خطأ واضحة باللغة العربية
