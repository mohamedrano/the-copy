# سياسة الأمان - The Copy

## الإبلاغ عن الثغرات الأمنية

نحن نأخذ الأمان على محمل الجد. إذا اكتشفت ثغرة أمنية، يرجى إبلاغنا عنها بشكل مسؤول.

### كيفية الإبلاغ
- **البريد الإلكتروني:** security@the-copy.com
- **GitHub Security Advisories:** [الإبلاغ عن ثغرة](https://github.com/your-org/the-copy/security/advisories/new)
- **لا تستخدم Issues العامة** للإبلاغ عن الثغرات الأمنية

### ما نتوقعه منك
- **الإبلاغ المسؤول** - لا تشارك التفاصيل علناً حتى نتمكن من إصلاحها
- **التفاصيل الكافية** - قدم وصفاً مفصلاً للثغرة
- **خطوات إعادة الإنتاج** - ساعدنا في فهم كيفية استغلال الثغرة
- **التحقق من التحديثات** - تأكد من أن الثغرة لم يتم الإبلاغ عنها مسبقاً

### ما يمكنك توقعه منا
- **الرد السريع** - سنرد خلال 48 ساعة
- **التحديثات المنتظمة** - سنبقيك على اطلاع بالتقدم
- **الاعتراف** - سنعترف بمساهمتك إذا رغبت
- **الإصلاح السريع** - سنعمل على إصلاح الثغرة بأسرع وقت ممكن

## إرشادات الأمان للمطورين

### 1. معالجة البيانات الحساسة

#### مفاتيح API
```typescript
// ❌ خطأ - لا تكشف المفاتيح في الكود
const API_KEY = 'sk-1234567890abcdef';

// ✅ صحيح - استخدم متغيرات البيئة
const API_KEY = process.env.VITE_GEMINI_API_KEY;
```

#### البيانات الشخصية
```typescript
// ❌ خطأ - لا تخزن بيانات حساسة في localStorage
localStorage.setItem('userPassword', password);

// ✅ صحيح - استخدم التخزين الآمن
sessionStorage.setItem('userSession', encryptedSession);
```

### 2. تنظيف المدخلات

#### HTML Content
```typescript
import DOMPurify from 'dompurify';

// ❌ خطأ - عرض HTML مباشرة
<div dangerouslySetInnerHTML={{ __html: userInput }} />

// ✅ صحيح - تنظيف HTML
<div dangerouslySetInnerHTML={{ 
  __html: DOMPurify.sanitize(userInput) 
}} />
```

#### Validation
```typescript
// ❌ خطأ - عدم التحقق من المدخلات
function processText(text: string) {
  return text.toUpperCase();
}

// ✅ صحيح - التحقق من المدخلات
function processText(text: string) {
  if (typeof text !== 'string' || text.length > 10000) {
    throw new Error('Invalid input');
  }
  return text.toUpperCase();
}
```

### 3. حماية من XSS

#### React Components
```typescript
// ❌ خطأ - عرض نص غير آمن
function UserMessage({ message }: { message: string }) {
  return <div>{message}</div>;
}

// ✅ صحيح - تنظيف النص
function UserMessage({ message }: { message: string }) {
  const cleanMessage = DOMPurify.sanitize(message);
  return <div>{cleanMessage}</div>;
}
```

### 4. حماية من CSRF

#### API Calls
```typescript
// ❌ خطأ - عدم استخدام CSRF tokens
fetch('/api/update', {
  method: 'POST',
  body: JSON.stringify(data)
});

// ✅ صحيح - استخدام CSRF tokens
fetch('/api/update', {
  method: 'POST',
  headers: {
    'X-CSRF-Token': getCsrfToken(),
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(data)
});
```

### 5. تشفير البيانات

#### البيانات الحساسة
```typescript
// ❌ خطأ - تخزين بيانات غير مشفرة
localStorage.setItem('userData', JSON.stringify(userData));

// ✅ صحيح - تشفير البيانات
const encryptedData = encrypt(JSON.stringify(userData));
localStorage.setItem('userData', encryptedData);
```
