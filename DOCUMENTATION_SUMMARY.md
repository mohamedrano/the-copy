## السكربتات المتاحة

### سكربتات التوثيق
```bash
npm run doc:build      # بناء التوثيق
npm run doc:lint       # فحص جودة التوثيق
npm run doc:coverage   # حساب تغطية التوثيق
```

### سكربتات الجودة
```bash
npm run type-check     # فحص الأنواع
npm run lint          # فحص الكود
npm run type:coverage # تغطية الأنواع
npm run linkcheck     # فحص الروابط
```

### سكربتات شاملة
```bash
npm run docs:verify   # فحص التوثيق الشامل
npm run ci:docs       # فحص CI/CD كامل
```
